## 📖 About This Repository

This repo contains **handcrafted notes** — written by hand, scanned/photographed, and organized as PDFs — covering the core architectures that underpin modern deep learning and AI.

Each note dives into:
- **Intuition** behind the architecture
- **Mathematical formulation** and derivations
- **Diagrams** to visualize data flow and key components
- **Key papers** and concepts referenced during study

These are not auto-generated or copy-pasted summaries. Every page has been written from scratch after reading original research papers and course material.

## 🧠 Topics Covered

### Sequence Models
- **RNN** — Vanishing gradient problem, unrolled computation graphs, BPTT
- **LSTM** — Cell state, forget/input/output gates, gating mechanisms

### Attention & Transformers
- **Attention Mechanism** — Scaled dot-product attention, multi-head attention
- **Transformer Architecture** — Encoder-decoder structure, positional encoding, self-attention
- **Flash Attention** — Memory-efficient attention computation, IO-aware algorithm

### Generative Models
- **Diffusion Models** — Forward/reverse diffusion process, DDPM, noise schedules, score matching

### Modern State Space Models
- **SSM & Mamba** — Structured State Space Models, selective scan mechanism, Mamba vs Transformer comparison

### Representation & Transfer Learning
- **SimCLR** — Contrastive learning framework, data augmentation strategy, NT-Xent loss
- **Transfer Learning** — Fine-tuning strategies, feature extraction, domain adaptation
